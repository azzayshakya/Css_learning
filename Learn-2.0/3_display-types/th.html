📌 Understanding the Different Display Types

Display Type Example Elements Characteristics
Block <div>, <p>, <form>, <h1>
  • Starts on a new line.
  • Takes full available width.
  • Can have width, height, margin, and padding.
  • Commonly used for structural elements.
Inline <span>, <a>, <strong>
  • Stays in the same line with adjacent elements.
  • Width and height are ignored; size depends on content.
  • Only padding and margin (left/right) are respected.
  • Used for styling specific words or phrases.
Inline-Block <button>, <input>, <img>
  • Behaves like inline (stays in the same line).
  • Respects width and height properties.
  • Allows for inline elements with box-like behavior.
  • Useful for buttons, form elements, and images.
Flexbox <div class="flex-container">
  • One-dimensional layout system (row or column).
  • Provides flexible alignment and spacing.
  • Uses properties like justify-content and align-items.
  • Best for creating responsive UI elements.
Grid <div class="grid-container">
  • Two-dimensional layout system.
  • Allows rows and columns with precise control.
  • Uses grid-template-rows and grid-template-columns.
  • Best for creating complex layouts with precise alignment.
Table <table>
  • Designed for tabular data representation.
  • Organizes content into rows and columns.
  • Consists of elements like thead, tbody, tr, th, and td.
  • Not recommended for layout design.
