ðŸ”¹ What Are CSS Variables?

CSS Variables (also known as Custom Properties) allow you to store reusable values in a stylesheet.

:root {
    --primary-color: #3498db;
    --text-size: 16px;
}

p {
    color: var(--primary-color);
    font-size: var(--text-size);
}

ðŸ”¹ Benefits of Using CSS Variables

  • Improves maintainability and reusability.
  • Helps in maintaining a consistent design system.
  • Allows for easy theme switching.
  • Can be modified dynamically using JavaScript.

ðŸ”¹ Defining and Using CSS Variables

CSS variables are declared inside the :root pseudo-class for global access.

:root {
    --main-bg: #f8f9fa;
    --font-color: #333;
}

body {
    background-color: var(--main-bg);
    color: var(--font-color);
}

ðŸ”¹ Local vs Global Variables

Variables inside :root are global, whereas local variables are specific to an element.

.container {
    --container-color: #e74c3c;
}

.container p {
    color: var(--container-color);
}

ðŸ”¹ CSS Variables and JavaScript

CSS variables can be manipulated dynamically using JavaScript.

document.documentElement.style.setProperty('--primary-color', '#e74c3c');
