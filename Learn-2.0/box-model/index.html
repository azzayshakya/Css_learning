CSS Box Model Documentation

1. What is the CSS Box Model?

The CSS Box Model is a fundamental concept in web design that describes how elements are displayed as rectangular boxes in a webpage. Every HTML element consists of:

  • Content: The actual content inside the box (text, images, etc.).
  • Padding: Space between the content and the border.
  • Border: A boundary around the element.
  • Margin: Space outside the border, separating elements.

4. Box Sizing

By default, the width and height apply only to the content. If padding and border are added, the box grows in size. To prevent this, we use box-sizing: border-box; which includes padding and border within the specified width.
